## What's needed before tagging v2.0.0

Set 5 GHA secrets so the release workflow can sign + notarize:

```bash
# Apple Developer ID Application certificate (.p12 from Keychain Access, base64-encoded)
base64 -i cert.p12 | gh secret set MACOS_CERT --repo CraigVG/mac2imgur

# Password used when exporting the .p12
gh secret set MACOS_CERT_PASSWORD --repo CraigVG/mac2imgur

# Apple ID email
echo "craigvandergalien@gmail.com" | gh secret set APPLE_ID --repo CraigVG/mac2imgur

# App-specific password from appleid.apple.com -> Sign-In -> App-Specific Passwords
gh secret set APPLE_APP_PASSWORD --repo CraigVG/mac2imgur

# 10-character Team ID from developer.apple.com -> Account -> Membership
gh secret set APPLE_TEAM_ID --repo CraigVG/mac2imgur
```

Verify with `gh secret list --repo CraigVG/mac2imgur`. Should show 6 secrets total (5 above plus the existing `SPARKLE_PRIVATE_KEY`).

## To ship v2.0.0

```bash
# 1. Manual smoke test - run docs/release-smoke-test.md against the local Release build:
#    ~/mac2imgur/build/Build/Products/Release/mac2imgur.app

# 2. Tag and push
cd ~/mac2imgur
git tag -a v2.0.0 -m "v2.0.0 - Modernized fork, Apple Silicon native"
git push origin v2.0.0

# 3. Watch the release workflow (~10 min)
gh run watch --repo CraigVG/mac2imgur

# 4. Verify the release
gh release view v2.0.0 --repo CraigVG/mac2imgur
gh release download v2.0.0 --repo CraigVG/mac2imgur --pattern '*.zip'
unzip mac2imgur.zip
xattr -p com.apple.quarantine mac2imgur.app  # should be empty if notarized correctly
```

## Known follow-ups (not blockers for v2.0.0)

- **Existing 2018 install will be replaced when v2.0.0 is dragged to /Applications.** Preferences and OAuth state preserved (same Bundle ID, same UserDefaults keys).
- **Existing 2018 build cannot auto-update to v2.0.0** because Sparkle keys changed (DSA → EdDSA). User must manually download v2.0.0 once. Updates from v2.0.0 → v2.0.1 onward are automatic.
- **Apple Developer secrets need user action** — can't be set inline by Claude.
- **SSH keys for git** not configured on this Mac — currently using HTTPS. Optional improvement.
- **No telemetry / crash reporting** by design. Crashlytics was removed; no replacement added (YAGNI for a personal-use revival).
